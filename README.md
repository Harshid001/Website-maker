# ShopCraft Studio

AI-powered no-code creative platform for creating websites, 2D designs, animations, and 3D models.

## Project Structure

- `frontend/`: React + Vite + Tailwind CSS
- `backend/`: Node.js + Express + MongoDB

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
